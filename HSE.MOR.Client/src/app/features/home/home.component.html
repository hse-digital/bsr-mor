Make a mandatory occurrence report

Use this service to make a mandatory occurrence report to the Building Safety Regulator (BSR). Make a mandatory occurrence report if you are:

Occupied buildings

If you are an accountable person or principal accountable person, you must make a mandatory occurrence report for an occupied building if something causes or if something risks

Buildings in design or construction

If you are a principal contractor or principal designer, you must make a mandatory occurrence report if something or if something risks You do not need to make a mandatory occurrence report for a design until the project has a building control application number.

Before you start

To make a mandatory occurrence report, we will ask you: Read the guidance on what occurrences should be reported. (opens in a new tab) If you are not a principal accountable person, accountable person, principal contractor or principal designer and want to raise an issue with BSR, you should make a complaint to BSR. A criminal offence is committed if a mandatory occurrence report is not made within 10 days of you or your organisation becoming aware of an occurrence or risk.

If you can't use this service

If you cannot use the online service, call BSR on 0300 790 6787. BSR is open Monday to Friday from 8.30am to 5pm, except Wednesdays when we are open from 10am to 5pm. A translation service is available if you would prefer to speak in a language other than English. Start now
